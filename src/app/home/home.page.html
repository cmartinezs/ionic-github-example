GitHub Desktop

GitHub Desktop es una aplicación de escritorio que le permite trabajar con GitHub desde su computadora.

GitHub Desktop se sincroniza con su cuenta de GitHub y le permite clonar repositorios, crear ramas, realizar cambios, confirmarlos y enviarlos a GitHub.

GitHub Desktop es una excelente herramienta para trabajar con GitHub, especialmente si no está familiarizado con la línea de comandos.
